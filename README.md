# Yathartha Labs Website
This repository contains the official website of **Yathartha Labs**, showcasing advanced drone technology solutions used in **Agritech, Defence, Education**, and **MultiUtility** sectors.
## Features
- Modern and responsive design
- High-quality background drone videos
- Clean UI for showcasing different drone categories
- Lightweight HTML, CSS, and JavaScript structure
## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages for hosting
## Purpose
This website is developed to represent the work, research, and drone solutions created at **Yathartha Labs**.
